# Paróquia Manager - Frontend

Interface moderna e intuitiva para o gerenciamento de paróquias, desenvolvida com **Next.js** e **React 19**.

## ✨ Funcionalidades

- **Dashboard de Catequese:** Visualização rápida de métricas e turmas.
- **Controle de Frequência:** Chamada digital com suporte a encontros marcados.
- **Gestão de Pastorais:** Criar e editar pastorais, vincular coordenadores e gerenciar identidade visual.
- **Portal de Eventos:** Publicação de notícias e eventos gerais ou específicos de cada pastoral.
- **Social & Localização:** Links para Instagram e definição de locais de encontro por pastoral.
- **Prisma & Postgres:** Banco de dados relacional com histórico de migrações.
- **Responsividade:** Design otimizado para celulares e computadores.
- **Acesso Admin:** Link rápido no cabeçalho para criação de eventos e gestão global.

## 🚀 Como acessar

Acesse a interface de usuário em:
`http://localhost:3000` (em ambiente de desenvolvimento).
